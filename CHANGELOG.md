# Changelog

All notable changes to **Family Tree** are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

---

## [1.0.0] - 2026-03-31

### Added
- Initial release
- Multi-generation family tree rendering with couple connectors
- Add / edit / delete family members
- Upload photos per member (base64 via FileReader API)
- Add spouse to any single member
- Add children under any member
- Role badges: Patriarch, Matriarch, Parent, Child
- Gender-coded card borders (blue / pink / purple)
- Auto-save to `localStorage`
- Export tree as JSON
- Import tree from JSON
- Reset to sample Anderson family
- Stats bar (total members, generations, couples)
- Dark glassmorphism UI with gold & coral gradient accents
- Responsive layout with horizontal scroll for wide trees
